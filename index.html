September 2018 - January 2019

Facial Recognition Program

Language: GNU Octave Version 3.8.0

Over my 1A term I developed a simple facial recognition software using machine learning algorithms that I learnt from Stanford University's Machine Learning coursera course.

The program is trained on hundreds of photos of Tom Holland and other celebrities. It's main function is to be able to tell whether or not a given picture is a picture of Tom Holland.

It uses a Gradient Descent Algorithm with Regularization and it achieves about 90% accuracy on my Test and Cross Validation Sets. The value for the regularization term Lambda and the dimension of the photo used were chosen based on the generated graphs to the left.

Box2D Physics Simulations

Language: Processing 3

In highschool physics classes I often wanted digital simulations for experiments. Sometimes there were no adequate simulations online so I learnt Processing 3 and Box2D to make them myself.

First Video: This simulation demonstrates how an initially chaotic system with planets orbiting in different directions evenentually stabilizes. Planets all fall into stable orbits all in the same direction.

Second Video: Here is a spring coupled pendulum system. As one pendulum oscillates, energy is transfered to the second pendulum via the spring. Eventually, all energy is transfered and the first pendulum becomes motionless. This process repeats continuously.

Third Video: This is a program that I made for fun and to test out and learn some of Box2D's features.

April 2016 - June 2016

Aquatic Attack Game

Language: Python 2.7 with Pygame

Aquatic Attack was developed using object oriented programming to manage the player, projectile, and environment classes. Using object oriented programming allowed us to inherit the from pygame sprite class and process the necessary collision events efficiently.

In this game one player controls the ranged Seahorse and another controls the melee Pufferfish. The Pufferfish is a lot faster but the Seahorse is much more maneuverable.

Other game elements include powerup fish swimming by, the wrap-around map, and a pillar which the Seahorse can't shoot through.
